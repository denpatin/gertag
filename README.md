Tagger for German language
==========================

Based on Stanford PoS Tagger.

## How to Run

1. `git clone https://github.com/denpatin/gertag.git && cd gertag`
* `bundle install`
* `./setup.sh`
* `rerun 'rackup'`
* Go to `http://localhost:9292`
